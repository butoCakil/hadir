📋 Dokumentasi Presensi PKL via WhatsApp

SMK Negeri Bansari – Tahun 2025

1. Pendahuluan

Sistem Presensi PKL ini memudahkan siswa dan pembimbing melakukan presensi, melihat rekap, dan mengakses informasi PKL langsung melalui WhatsApp.
Layanan ini memiliki beberapa menu dan perintah yang bisa digunakan sesuai kebutuhan pengguna.
Pesan (Chat) akan dibalas otomatis oleh bot, pastikan perintah yang dikirimkan sesuai.

2. Daftar Perintah Utama

2.1 Perintah Umum

Perintah ini dapat diakses oleh siapapun, meskipun nomor belum terdaftar di sistem.

Perintah Fungsi
info Menampilkan menu utama layanan.
cek Melihat status nomor WA yang digunakan.
cek <nis/noHP> Melihat status berdasarkan NIS atau nomor HP.
reg <nis> Mendaftarkan nomor WA ke sistem.
admin Kontak Admin
help Download Panduan Dokumentasi Penggunaan
set <NIS> <NomorHP> Reset nomer WA -> kendala: Jika nomor tidak direspon Sistem

2.2 Perintah untuk Siswa

Perintah ini hanya dapat diakses oleh nomor yang telah terdaftar sebagai siswa.

Perintah Fungsi
reg <nis> Mendaftarkan nomor WA ke sistem.
unreg Menghapus nomor WA dari sistem.
masuk Melakukan presensi harian masuk ⇒ Wajib menyertakan Foto
izin Melakukan presensi harian izin
sakit Melakukan presensi harian sakit
libur Melakukan presensi harian libur
batal Membatalkan presensi.
lupa Presensi untuk hari sebelumnya (maks. 2x sehari).
rekap Mengetahui rekap presensi.

2.3 Perintah untuk Pembimbing

Perintah ini hanya dapat diakses oleh nomor yang telah terdaftar sebagai pembimbing. Nomor pembimbing akan didaftarkan oleh admin.

Perintah Fungsi
cek rekap Lihat rekap presensi per kelas/pembimbing/DUDI.
cek rekap <nis/noHP> Lihat rekap presensi individu.
cek rekap <kelas> Lihat rekap presensi per kelas.
cek presensi Lihat Statistik Presensi.
cek presensi <nis/noHP> Lihat presensi harian individu.
input <nis/noHP> Input presensi hari ini untuk siswa.
input <nis/noHP> <tgl_awal> <tgl_akhir> Input presensi dalam rentang tanggal.
cari <keyword> Cari data siswa/pembimbing/DUDI.

3. Panduan Penggunaan

3.1 Mengakses Menu Utama

  • Kirim perintah: info
  • Balas dengan nomor menu sesuai pilihan.
    • Contoh: ketik 2 untuk fitur Lupa Presensi.

3.2 Registrasi Nomor WA

  • Ketik: reg <nis>
    • Contoh: reg 1234
  • Sistem akan menampilkan data.
  • Balas ya untuk mendaftar, atau tidak untuk mengulang dengan NIS lain.

3.3 Presensi Harian

  • Pastikan nomor WA sudah terdaftar.
  • Ambil foto selfie di lokasi PKL.
  • Kirim foto dengan format:
    <keterangan> <catatan>
  • Pilihan keterangan: masuk, izin, sakit, libur
  • Tunggu balasan: ✅ Presensi Berhasil

3.4 Lupa Presensi

  • Akses menu info → ketik 2
  • Kirim foto selfie seperti biasa.
  • Gunakan format:
    lupa <keterangan> <tanggal> <catatan>
  • Tunggu balasan: ✅ Presensi "Lupa" Diterima
  • Maksimal 2 kali per hari.

3.5 Membatalkan Presensi

  • Untuk membatalkan presensi sendiri:
    batal <tanggal>
  • Untuk membatalkan presensi siswa (oleh Admin):
    batal <no_wa_siswa> <tanggal>

3.6 Melihat Rekap Presensi

  • Akses menu info → ketik 6
  • atau ketik rekap
  • Sistem akan menampilkan data rekap.
  • Link detail akan diberikan.

3.7 Kontak Admin

  • Ketik: admin
  • Atau akses menu info → ketik 7
  • Tanpa mengakses perintah ini, pesan mungkin akan diabaikan

4. Keterangan Simbol Rekap

Simbol Arti
✅ Masuk
🔵 (Biru) Izin
🟡 (Kuning) Sakit
🔴 (Merah) Libur
➖ Libur Weekend
❌ Tidak presensi

5. Fitur pengingat presensi (siswa)

  • Fitur ini akan berjalan otomatis pada jam tertentu setiap hari.
  • Jika ada yang belum presensi akan mendapatkan pesan ini.
  • Jika mendapatkan pesan ini cukup konfirmasi sesuai pilihan.
    • ya
      ⇒ Balas dengan ini, akan otomatis tersimpan / tercatat sebagai libur
    • tidak
      ⇒ Harus tetap melakukan presensi seperti masuk seperti biasa (dengan foto)
    • sakit
      ⇒ Balas dengan ini, akan otomatis tersimpan / tercatat sebagai sakit
    • izin
      ⇒ Balas dengan ini, akan otomatis tersimpan / tercatat sebagai izin
    • libur
      ⇒ Balas dengan ini, akan otomatis tersimpan / tercatat sebagai libur

6. Notifikasi rekap presensi (pembimbing)

  • Setiap satu minggu sekali, pembimbing akan mendapatkan notifikasi rekap presensi siswa bimbingannya.
  • Data ini lengkap dengan Nama Siswa, DUDI, Kelas, NIS dan Nomor HP
  • Data Rekap adalah hasil presensi selama 7 hari sebelumnya.

7. Download Panduan Penggunaan Layanan

  • Untuk mendapatkan Dokumentasi panduan penggunaan chatbot Whatsapp Presensi PKL, kirim perintah: help

8. Catatan Penting

  • Fitur Lupa Presensi hanya untuk hari sebelumnya.
  • Semua aktivitas tercatat oleh sistem.
  • Gunakan format penulisan yang benar agar sistem dapat memproses perintah.
  • Ketika ingin bertanya pastikan beri perintah admin terlebih dahulu. Jika tidak, kemungkinan pesan akan diabaikan.
